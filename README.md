# 🌓 Mini Layout con Cambio de Tema

Pequeño proyecto frontend que permite alternar entre **modo claro** y **modo oscuro** con un solo clic.

## 🚀 Demo

- Modo claro por defecto
- Botón que cambia el tema y su propio texto
- Diseño simple con navbar y contenido principal

## 📁 Estructura
├── index.html
├── styles.css
└── script.js

## 🛠️ Tecnologías

- HTML5
- CSS3
- JavaScript (Vanilla)

## ▶️ Uso

1. Clona el repositorio
2. Abre `index.html` en tu navegador
3. Haz clic en el botón para cambiar entre tema claro/oscuro

## 🎨 Personalización

Puedes modificar los colores en `styles.css`:
- Tema claro: fondo `#f5f5f5`, navbar `#007bff`
- Tema oscuro: fondo `#121212`, navbar `#222`

# 🛍️ SportMax - Catálogo de Productos

Mini tienda deportiva con catálogo dinámico de productos, componentes reutilizables (header/footer) y Web Components.

## ✨ Características

- Header y footer dinámicos cargados con `fetch()`
- Tarjetas de producto con **Web Components** (`<product-card>`)
- Catálogo de productos deportivos (chaquetas, tenis, shorts, camisetas)
- Diseño responsive
